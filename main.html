Let's start off with background-color:; It's really simple all it does is color the background of the object you assign it to such as if you have:

body {
background-color: white;
}

You will then have a white background for your web page although you can assign it to other tags like h1 tags example:

h1 {
background-color: red;
}

Then you will have a red background for your h1 tag So basically you can put background-color:; in any CSS styling! It's that simple!

dashed

Borders are also pretty simple to use for instance here is how to use borders in CSS. Depending on the id you used in your HTML (we will learn about it in another lesson)

#dashed{
border:dashed;
}
#dotted{
border:dotted;
}
#inset{
border:inset;
}
#outset{
border:outset;
}
#solid{
border:solid;
}
#ridge{
border:ridge;
}
#groove{
border:groove;
}

Now lets show you how to color borders well thats simple you use border-color:; example:

#border-color-example{
border:solid;
border-color:#ff3333;
}

A red border

One lest thing rounding the corners using border-radius:; example below:

#border-radius-example{
border:solid;
border-color:#ff3333;
border-radius:10px;
}

A red and rounded border
